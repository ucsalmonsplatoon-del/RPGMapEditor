# 🗺️ RPGマップエディタ - 使い方ガイド

## 概要
このツールは、RPGツクールMZ用のマップ作成補助ツールです。
タイルセット画像から自由に部品を切り出して、マップ上に配置・編集できます。

---

## 📋 起動方法

1. **rpg_map_editor.html** をダブルクリックしてブラウザで開く
2. Chrome、Firefox、Edge などのモダンブラウザ推奨

---

## 🎮 基本的な使い方

### 1️⃣ 画像の読み込み

#### 方法1: ボタンクリック
- **📁 タイルセット読込** - 左側にタイルセット画像を読み込み
- **🗺️ マップ画像読込** - 右側に背景マップを読み込み

#### 方法2: ドラッグ&ドロップ
- 画像ファイルを直接エリアにドロップするだけ！
- 左エリア → タイルセット
- 右エリア → マップ背景

### 2️⃣ タイル選択と配置

1. **左側のタイルセット**でマウスドラッグして範囲選択
2. 選択した部分が青い枠で囲まれる
3. **配置モード**の状態で右側のマップをクリック
4. クリックした位置に選択したタイルが配置される

### 3️⃣ オブジェクトの編集

1. **編集モード**に切り替える
2. 配置したオブジェクトをクリックで選択
3. 操作:
   - **ドラッグ** → 移動
   - **四隅のハンドル** → リサイズ
   - **Deleteキー** → 削除

---

## 🖱️ マウス操作

| 操作 | 機能 |
|------|------|
| **左クリック** | 選択・配置 |
| **中ボタン押しながらドラッグ** | 画面移動（パン） |
| **ホイール回転** | ズーム |
| **ドラッグ** | 範囲選択（タイルセット）/ オブジェクト移動（編集モード） |

---

## ⌨️ キーボードショートカット

| キー | 機能 |
|------|------|
| **Ctrl + Z** | 元に戻す |
| **Ctrl + Y** | やり直し |
| **Delete** | 選択オブジェクトを削除 |
| **Ctrl + +** | ズームイン |
| **Ctrl + -** | ズームアウト |
| **Ctrl + 0** | ズームリセット |

---

## 💾 プロジェクト管理

### 保存
1. **💾 プロジェクト保存** をクリック
2. `.json`ファイルとして保存される
3. すべての配置情報と画像データを含む

### 読み込み
1. **📂 プロジェクト読込** をクリック
2. 保存した`.json`ファイルを選択
3. 前回の作業状態が完全に復元される

### PNG出力
1. **🖼️ PNG出力** をクリック
2. 配置したオブジェクトが合成された最終画像を保存

---

## 🎯 活用例

### タイルセット作成
1. RPGツクールのタイルセット画像（Outside_B.pngなど）を左に読み込み
2. 必要な部品を選択して新しいマップに配置
3. PNG出力して新しいタイルセットとして使用

### マップデザイン
1. ベースとなるマップ画像を右側に読み込み
2. タイルセットから装飾オブジェクトを選択
3. 自由に配置してマップをデザイン
4. 完成したらPNG出力

### 看板・オブジェクト配置
1. 自作の看板画像をタイルセットとして読み込み
2. 48x48ピクセル単位で切り出し可能
3. 大きな看板は複数タイルとして配置

## 💡 Tips

### ズーム機能の活用
- タイルセット側もズーム可能
- 細かい部分を正確に選択したい時に便利
- マウスがある側のエリアがCtrlキーでズーム

### 効率的な作業
- プロジェクト保存で途中経過を保存
- Ctrl+Zで失敗を即座に取り消し
- 中ボタンパンで素早く画面移動

### 大きなオブジェクトの扱い
- 96x96や144x144など、48の倍数サイズが扱いやすい
- 分割されたタイルは隙間なく配置される
- リサイズ機能で後から調整可能

---

## ⚠️ 注意事項

- ブラウザの機能を使用するため、インターネット接続は不要
- プロジェクトファイルは画像データを含むため、サイズが大きくなる場合がある
- 作業中は定期的にプロジェクト保存を推奨

---

## 🔧 トラブルシューティング

### 画像が読み込めない
- PNG、JPG、GIF形式に対応
- ファイルサイズが大きすぎる場合は縮小してから使用

### オブジェクトが選択できない
- 「編集モード」になっているか確認
- オブジェクトが重なっている場合は、リストから削除

### ズームがおかしい
- Ctrl+0でリセット
- ブラウザの表示倍率も100%に設定
